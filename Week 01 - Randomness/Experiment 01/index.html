Experiment 1 - Like, literally, dude

Hello!

Description

I started a book by sociolinguist Valerie Fridland appropriately called "Like, Literally, Dude" -- it takes a deep dive into language as both function and fashion.

I wanted to see how readible -- or unreadible -- text became in, like, bad english.

Technical

The template already provided a basic template. The original piece generated fantasy hero callings via plugging keys from several different banks into designated flagged spots.

I myself just created a new bank with "bad english" keys and integrated new flagged spots for this bank.
